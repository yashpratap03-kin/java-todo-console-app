# Java Console-Based To-Do List App

A simple **console-based To-Do List application** developed in **Java** as part of a Java Developer Internship Task (Week 1).  
The application allows users to manage tasks using a menu-based interface with persistent file storage.

 # Technologies Used
- Java (Core)
- ArrayList
- File Handling
- OOP Concepts
- Command Line (CLI)

# Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved into `tasks.txt` automatically
- Loads saved tasks on startup

# How to Run

# Requirements
- Java JDK installed on system

# Steps
javac ToDoApp.java
java ToDoApp
